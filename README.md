# Claude Skills

Colección personal de skills, plugins y commands para Claude Code.

## Estructura

```
claude-skills/
├── skills/        # Skills personalizados
├── plugins/       # Plugins MCP
└── commands/      # Commands reutilizables
```

## Uso

### Skills

Copia el skill deseado a tu configuración de Claude Code:

```bash
# Linux/macOS
~/.claude/skills/

# Windows
%USERPROFILE%\.claude\skills\
```

### Plugins

Los plugins MCP se configuran en `settings.json` de Claude Code.

## Contribuir

Si quieres agregar un skill o plugin:

1. Fork este repositorio
2. Crea tu skill/plugin en la carpeta correspondiente
3. Documenta su uso en un README dentro de la carpeta
4. Abre un PR
