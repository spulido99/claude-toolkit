## Usage Tips

### Matching Hooks to Content

| Hook Type | Best For | Expected Engagement |
|-----------|----------|-------------------|
| Number + Outcome | Credibility, concrete value | High saves |
| Contrarian | Discussion, positioning | High comments |
| Curiosity Gap | Click-through, reach | High impressions |
| Vulnerable/Story | Connection, trust | High engagement |
| Question | Discussion starter | High comments |
| News + Opinion | Timeliness | High shares |
| Pattern Interrupt | Standing out | Variable |

### Customization Tips

1. Replace placeholder numbers with your real data
2. Add specific company/industry context
3. Time-peg hooks to recent news when possible
4. Test variations of the same hook structure
5. Track which patterns perform best for your audience
