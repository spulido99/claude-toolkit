# Digital Marketing Plugin

A Claude Code plugin that helps users with little or no marketing knowledge define, create, execute, measure, and improve digital marketing campaigns.

## Features

- **Guided Campaign Definition**: Interactive workflow to define objectives, audience, and strategy
- **Content Generation**: Platform-optimized content in English and Spanish
- **Automated Posting**: Post to social media via Chrome browser automation
- **Analytics Integration**: Check Meta Business Suite and Google Analytics
- **Optimization Recommendations**: Data-driven suggestions to improve campaigns

## Requirements

- Claude Code with Chrome browser extension (claude-in-chrome)
- Access to your social media accounts via Chrome

## Commands

| Command | Description |
|---------|-------------|
| `/define-campaign` | Interactive interview to define your campaign strategy |
| `/create-content` | Generate content for specific platforms |
| `/post-social` | Post content to social media via Chrome |
| `/check-analytics` | Open and analyze platform analytics |
| `/improve-campaign` | Get optimization recommendations |
| `/campaign-status` | View current campaign summary |

## Data Storage

Campaign data is stored locally in `.claude/digital-marketing.local.md` within your project directory.

## Supported Platforms

- Instagram
- Facebook
- LinkedIn
- Twitter/X
- TikTok

## Analytics

- Meta Business Suite (Instagram + Facebook)
- Google Analytics
