# DeepAgents Builder

Build production-ready AI agents with LangGraph. This plugin provides skills, agents, and commands for designing, implementing, testing, and evolving agent systems.

## Skills

| Skill | Description |
|-------|-------------|
| **quickstart** | Getting started with LangGraph agents |
| **architecture** | Agent topologies and bounded contexts |
| **patterns** | System prompts, tool design, anti-patterns |
| **tool-design** | AI-friendly tool design principles |
| **evals** | Testing, benchmarking, debugging |
| **evolution** | Maturity model and refactoring |

## Agents

| Agent | Description |
|-------|-------------|
| **agent-architect** | Designs agent topologies based on requirements |
| **code-reviewer** | Reviews agent code for anti-patterns and security issues |
| **tool-architect** | Designs and generates AI-friendly tools |

## Commands

| Command | Description |
|---------|-------------|
| `/new-sdk-app` | Scaffold a new agent project |
| `/design-topology` | Interactive topology design |
| `/validate-agent` | Check agent code for anti-patterns |
| `/add-interactive-chat` | Generate a chat console for testing |

## Key References

The `api-cheatsheet.md` skill serves as the canonical API reference for LangGraph patterns used across all skills and agents.
